# 📜 Scripts

This folder contains all PowerShell scripts used in the lab.

| Script | Description |
|---|---|
| `bulk-user-creation.ps1` | Bulk user onboarding from CSV with group assignment |
| `dept-ou-groups.ps1` | Automated OU and security group creation per department |
| `password-policy.ps1` | Configure domain password and lockout policy |
| `arm-deploy.ps1` | Deploy Azure resources using ARM templates |

> All scripts use `-WhatIf` mode by default for safety. Remove it when ready to apply.
